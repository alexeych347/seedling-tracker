# Мой подоконник · seedling-tracker

Интерактивный помощник по домашней рассаде: трекер растений, напоминания о поливе, расчёт светового дня, библиотека сортов.

Чистый HTML/CSS/JS — без фреймворков, бандлеров и сервера. Данные хранятся в `localStorage`.

## Структура

- **`app.html`** — само приложение (светлая тема, стиль подоконника). Сцена неба, стеклянные панели (погода, световой день, напоминания), карточки горшков на подоконнике, статистика внизу.
- **`index.html`** — маркетинговый лендинг (светлая тема). Hero, «Как это работает», возможности, AI-блок, отзывы, тарифы.
- **`in-development.html`** — страница-заглушка для нереализованных функций (вход, тарифные планы).
- **`js/database.js`** — данные по **15 культурам** и 48+ сортам: дробные интервалы полива (в часах), стадии роста, советы по уходу.
- **`js/app.js`** — вся логика рендера и состояния. Один объект `State`, никакого virtual DOM.
- **`css/style.css`** — стили приложения (десктоп-first, с базовой мобильной адаптивностью).

## Как открыть

PowerShell:

```powershell
Start-Process app.html        # приложение
Start-Process index.html      # лендинг
```

Или просто откройте файл двойным кликом в браузере.

## Что умеет приложение

- **15 культур × 48+ сортов** с индивидуальным графиком полива и стадиями роста
- Дробные интервалы полива — от 18 ч (огурцы) до 60 ч (халапеньо, лук красный)
- Авто-напоминания о поливе с отображением оставшихся часов
- История полива — последние 20 поливов с датами (хранится в `localStorage`)
- Живая погода Москвы через Open-Meteo API (температура, влажность, давление, УФ-индекс, почасовой прогноз)
- Расчёт светового дня: восход, закат, длительность
- SVG-визуализация растений по стадиям (15 уникальных стилей)
- Единые карточки горшков: горшок + инфо на одной белой полупрозрачной плашке
- Глоссарий культур с подробной агротехникой
- Экспорт данных в JSON и сброс через страницу настроек
- Мобильная адаптация: боковое меню-оверлей, нижняя навигация
- Данные сохраняются в `localStorage` (ключи `rassada_v3`, `rassada_reminders`)

## Дорожная карта

- Виджет «Дней до сбора урожая» на обзорной странице
- Виджет «Мини-прогноз на неделю»
- AI-агроном (Anthropic API): персональные советы с учётом сорта, стадии и погоды
- Фото-диагностика болезней
- Bluetooth-датчики влажности и освещённости (Web Bluetooth API)
- Авторизация и платные тарифы — после стабилизации AI и датчиков
